A computer is device that accepts information (in the form of digitalized data) and manipulates it for some result based on a program or sequence of instructions on how data is to be processed. Complex computers also needs the means for storing data (including the program, which is also a form of data) for some necessary duration. A program may be invariable and built into the computer (and called logic circuitry as it is on microprocessors) or different program may be provided to the computer (loaded into its storage and then started by administrator or user). Today's computer have both type of programming.


An operating systems (OS) is a software program that manages computer hardware and software resources and provides common service for computer programs.
